linux常用命令

在微博上发现已有同学整理了一个不错的linux命令文档：http://115.com/file/c2rzrvw3#Unix-Linux 这篇文章等遇到好的命令再继续更新。。。

文件命令 | 程序命令 | 网络命令 | 系统命令

文件

创建一个文本文件

touch test.txt

删除文件

rm -f text.txt

创建一个目录

mkdir newdir

删除一个目录

rm -r dirname

显示目录中的文件信息

ls -lt

改变文件或目录的权限

chmod 666 test.txt

查看文件夹大小

du -h --max-depth=1 ./mp3

解压tar文件

tar zxpf test.tar.gz

程序

安装程序

显示正在运行的程序

ps -ef

关闭程序

kill -QUIT [PID]

强制关闭程序

kill -9 [PID]

网络

查看程序占用端口情况

netstat -anp

-a显示所有连接，-n显示端口情况，-p显示程序PID

系统

重启

reboot

关机

poweroff
